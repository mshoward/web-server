unsigned int DEMOGRAPHIC::demographic_id

Easiest way to compare demographics is to abstract them into numbers. e.g. race could be fields[0], min_age could be fields[1], etc.

Fields that don't matter can be 0 or less than 0.

There is difficulty in describing something like this in programming. An elegant, proper description will be the result of proper description in the database and communication between the people working on the database.

For now, I will implement a temporary cursory comparison function and craft a better one once the DEMOGRAPHIC architecture is more settled.
